Classifying Sleep-Dependent Behavioral Patterns with GNNs

L. Bastian, E.-M. Kurz, L. Gutjahr, H. Noack & J. Born
PyTorch Geometric NNConv GNNExplainer Python

Overview

Modeled toddlers' behavioral sequences as directed graphs (11 node types, transition-probability edges) and trained a 3-layer NNConv GNN to classify sleep vs. wake encoding trials. Achieved 65.8% accuracy (p=0.043); GNNExplainer identified key discriminative behavioral transitions.

Key Results

  • 3-layer NNConv with shared edge MLPs, ReLU6, combined max & mean pooling over 21 node features, edge probabilities, and 24 graph features
  • 73% sensitivity, 67% specificity validated with 5,000 permutations
  • GNNExplainer revealed "looking into container" → "container play" as key discriminative transition
  • Null at familiarization (50.0%, p=0.739) confirmed specificity to post-sleep behavioral changes

Method

Each behavioral sequence was encoded as a directed graph where nodes represent one of 11 behavioral categories and edge weights correspond to transition probabilities between consecutive actions. Node features (21-dimensional) captured frequency, duration, and sequential properties of each behavior type. Graph-level features (24-dimensional) described global sequence statistics.

The NNConv architecture processes these heterogeneous graphs through three message-passing layers with edge-conditioned neural networks, followed by combined max and mean graph-level pooling. Permutation testing (5,000 iterations) provided non-parametric significance, while GNNExplainer attributed classification decisions to specific subgraph motifs.
