Sleep Forms Flexible Context Representations in Toddlers

L. Bastian, E.-M. Kurz, L. Gutjahr, H. Noack & J. Born
PyTorch Geometric NNConv GNNExplainer Linear Mixed-Effects Models EEG Python

Overview

Sleep is known to consolidate episodic memory in adults, but whether and how it supports memory formation during early life — when the hippocampus is still structurally immature — remains unclear. In this study, we examined the effects of sleep on spatial context memory in 28 toddlers aged 2–3 years using a within-subject crossover design. Children were familiarized with two decorated rooms, followed by either a ~90-minute nap or an equivalent wake period, and then tested with a hide-and-seek game requiring them to relocate toys in the correct room context.

We combined behavioral analysis, EEG sleep recordings, and machine learning to comprehensively characterize how sleep shapes memory. Linear mixed-effects models tested condition differences in memory performance; spindle and slow-oscillation coupling analyses linked EEG sleep features to contextual memory; and a graph neural network was used to decode sleep-dependent behavioral signatures from the encoding phase.

Key Findings

  • Sleep selectively reduced context errors (confusing correct containers between rooms) compared to wakefulness, without affecting overall accuracy or random errors
  • Contextualized Memory Index (CMI) was higher after sleep (β = 0.15, p = 0.015), while decontextualized memory decreased — indicating that sleep promotes abstracted context representations
  • Spindle density during NonREM sleep predicted fewer context errors and higher CMI (β = 0.11, p = 0.012)
  • SO–spindle phase-amplitude coupling at 12–13 Hz predicted better contextualized memory (ρ = 0.534, p = 0.027), the first demonstration of this link in children this young
  • Sleep benefits were strongest for younger toddlers (25–30 months), suggesting a developmental window where wake-based context maintenance is particularly fragile

Graph Neural Network Analysis

To understand whether sleep reshaped how children interacted with the task, we scored each child’s behaviors during the encoding (hiding) phase across 10 behavioral categories — including interactions with toys, containers, and the experimenter. Transition probabilities between these behaviors were computed to build directed graphs for each trial.

A 3-layer NNConv GNN (PyTorch Geometric) was trained on these graphs to classify whether a trial occurred after sleep or wakefulness. The model used 21-dimensional node features (frequency, duration, sequential statistics), transition-probability edge weights, and 24 graph-level features. It achieved 65.8% accuracy (p = 0.043 over 5,000 permutations), with 73% sensitivity and 67% specificity.

Crucially, the GNN could not differentiate conditions at familiarization (50.0%, p = 0.739), confirming specificity to post-sleep changes. GNNExplainer revealed that the transition from “looking into container” to “container play” was the key discriminative feature: after wakefulness, children needed to physically interact with containers to recall the correct locations, whereas after sleep they appeared to rely on an internalized spatial representation — consistent with sleep-dependent memory abstraction.

Significance

This study provides some of the earliest evidence that sleep-dependent memory consolidation — mediated by canonical sleep oscillations — already operates in children as young as 2 years, despite hippocampal immaturity. The GNN analysis demonstrates how graph-based deep learning can uncover structured behavioral patterns that traditional statistical approaches would miss, providing interpretable insights into the cognitive effects of sleep.
