SODEEP: A Lightweight Causal CNN for Real-Time Slow Oscillation Detection

L. Bastian, T. Näher, J. Born & P. Fries
PyTorch Causal CNN LSTM Real-Time EEG
Work in Progress — Currently training multi-species foundation models

Overview

SODEEP is a lightweight causal convolutional neural network for real-time detection of slow oscillations (SOs) in EEG sleep data. The model predicts UP-to-DOWN and DOWN-to-UP state transitions in a sequence-to-sequence classification setup, operating under strict causality constraints with only ~100k parameters and 3 ms inference time.

Researchers studying the link between sleep and memory increasingly need causal manipulations of oscillatory sleep mechanisms. This requires real-time SO detection with minimal latency and high accuracy. Most existing methods rely on handcrafted algorithmic approaches with heavy filtering and fixed thresholds. SODEEP replaces these with a learned, end-to-end causal model.

Architecture and Training

The model takes causally band-pass filtered single-channel EEG (IIR, 1–25 Hz) as input and outputs per-timepoint class probabilities for baseline, UP-to-DOWN, and DOWN-to-UP states. The architecture uses stacked causal dilated convolutions for parameter-efficient large receptive fields, followed by LSTM layers for temporal context.

Training uses a custom focal transition loss that encodes biological constraints: a transition matrix penalizes impossible state sequences, while a temporal smoothness term penalizes large logit derivatives. Early stopping is based on focal loss, with 5 linear warm-up epochs followed by cosine decay over 100 epochs. Regularization includes dropout (0.1–0.15) and weight decay (0.001).

At inference time, a causal hysteresis decoder enforces biologically plausible state sequences: per-class entry thresholds and minimum dwell counts prevent spurious transitions. The first decoded sample of the DOWN-to-UP class serves as the stimulation trigger.

Significance and Ongoing Work

SODEEP replaces handcrafted algorithmic approaches for real-time sleep oscillation detection with a learned, end-to-end causal model that achieves dramatically higher accuracy at a fraction of the latency. This enables precise closed-loop stimulation experiments that were previously limited by detection delays.

We are currently scaling the approach toward multi-species foundation-type models for neural oscillation detection, training on large-scale datasets across species and recording modalities. The goal is a generalizable real-time model that transfers across invasive and non-invasive recordings, different oscillatory phenomena (spindles, ripples), and different species. In parallel, we are collaborating with the Ziemann lab at the University of Tübingen to implement SODEEP in a real-time TMS paradigm.
