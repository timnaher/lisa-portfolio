Riemannian Geometry Boosts fNIRS Brain-State Classification

T. Näher*, L. Bastian* et al.
Neurophotonics 12(4) →
SPD Manifolds pyRiemann fNIRS Block Diag. Kernels

Overview

Introduced Riemannian classification for functional near-infrared spectroscopy (fNIRS) using block diagonal SPD covariance kernels that jointly capture oxygenated (HBO) and deoxygenated (HBR) hemoglobin co-activation patterns. Achieved 89% two-class and 70% eight-class accuracy, substantially outperforming traditional feature-extraction approaches.

Key Results

  • Block diagonal SPD matrices combining HBO and HBR covariance structures into a joint kernel on the positive definite manifold
  • Multiple Riemannian classifiers benchmarked (MDM, tangent space LDA/LR/SVM) against traditional methods across 8 mental tasks
  • Wasserstein distance analysis validated that co-activation patterns are more similar within than between tasks
  • 89% two-class accuracy and 70% eight-class accuracy, demonstrating the power of geometric approaches for fNIRS

Method

Standard fNIRS classification pipelines extract hand-crafted features (mean, slope, peak) from hemodynamic time series and feed them to conventional classifiers. This work instead treats each trial's multi-channel HBO and HBR signals as a joint covariance structure, constructing block diagonal SPD matrices that preserve both within-chromophore and cross-chromophore spatial correlations.

These SPD matrices are classified on the Riemannian manifold using geodesic-based methods (Minimum Distance to Mean) or after tangent space projection with standard classifiers (LDA, logistic regression, SVM). The geometric approach captures richer spatial information than univariate features and naturally handles the positive-definiteness constraint of covariance data.
